# Дитячий магазин одягу

Цей проект є інтернет-магазином дитячого одягу. Ми прагнемо надавати широкий асортимент якісного одягу для дітей різного віку.

## Особливості
- Широкий вибір одягу для різних сезонів
- Зручний інтерфейс для покупок
- Безпечні способи оплати

## Інсталяція
1. Клонуйте репозиторій: `git clone https://github.com/mlutcenko-sys/dityachyy-shop`
2. Встановіть всі залежності: `npm install`
3. Запустіть проект: `npm start`

## Використання
- Перейдіть на головну сторінку, щоб переглянути товари.
- Додайте товари в кошик і оформте замовлення, слідуючи інструкціям на екрані.
